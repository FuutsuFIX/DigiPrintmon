How to use DigiPrintmon, a Digimon proxy generator:

In order to use this Digimon Proxy Maker you need to:
Enter a card list in the textbox.
Format for inserting cards:
Quantity Name Id

In case you are a digimoncard.app user, just check the box that is placed over the textbox!
This will change the format into
Id Name Quantity

The Name is optional in both cases.

You don't know how to make a list of cards for the Digimon Card Game 2020?
Use digimoncard.dev !
> Click Deck Builder
> Insert cards by clicking them
> In the Deck Builder click Import/Export
> make sure to be in Export and text mode
> Copy the text in the section and paste in the textbox here

Everything is ready,press download pdf and have fun !

Want to use your favourite artwork?

In order to choose an artwork for a card, follow these instruction:
You can write _P1, _P2 and ecc to choose any artwork after the card code.
For example:
1 Agumon BT1-010_P1
is going to use its first released alternative artwork.
